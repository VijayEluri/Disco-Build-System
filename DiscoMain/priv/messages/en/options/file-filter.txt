-f | --filter <filter1>:...

  Only show the files specified by the filter string, which is a
  colon-separated list of one or more filter specifications. Each filter
  specification describes the type of file you're interested in seeing. 
  Filter specifications are additive, so providing two filters results 
  in the union of the two being shown.
   
  Filter specifications can be one of:
  
#include options/path-spec.txt
